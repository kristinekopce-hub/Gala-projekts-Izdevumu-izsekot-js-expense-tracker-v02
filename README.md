Izdevumu izsekotājs (Python)

Vienkārša programma Python valodā personīgo izdevumu uzskaitei.
Programma ļauj lietotājam pievienot, apskatīt, filtrēt, apkopot, dzēst un eksportēt izdevumus.
Šis projekts demonstrē Python pamata prasmes, tostarp darbu ar failiem, datu apstrādi, programmas sadalīšanu moduļos un komandrindas lietotāja interfeisu.

Funkcionalitāte
Pievienot jaunus izdevumus (datums, summa, kategorija, apraksts)
Parādīt visus izdevumus tabulas veidā
Filtrēt izdevumus pēc mēneša
Skatīt kopsavilkumu pa kategorijām
Dzēst izdevumus pēc numura
Eksportēt izdevumus CSV failā
Saglabāt datus JSON failā

Projekta struktūra│ 
 - app.py       # Galvenā programma
 - storage.py   # Datu saglabāšana un ielāde (JSON)
 - logic.py     # Filtrēšana un aprēķini
 - export.py    # Eksports uz CSV
 - izdevumi.json # Saglabātie izdevumi
Tehnoloģijas un bibliotēkas
-Python 3
-JSON datu saglabāšanai
-CSV eksportam
-tabulate bibliotēka tabulu attēlošanai terminālī

Palaidiet programmu:
python app.py
